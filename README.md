# WhatsApp Service

Serviço Node.js para integração WhatsApp.

## Como rodar localmente

1. Instale as dependências:
   npm install
2. Configure as variáveis de ambiente (veja .env.example).
3. Rode o serviço:
   npm start

## Deploy no Render
- Use o Dockerfile incluso.
- Configure as variáveis de ambiente no painel do Render.
